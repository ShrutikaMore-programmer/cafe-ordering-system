# ☕ Shrutika's Cafe Ordering System

A Python command-line program that simulates a cafe ordering system.

## Features

* Displays a formatted cafe menu
* Allows users to order multiple items
* Calculates total price
* Generates a receipt

## Menu Categories

* Espresso
* Hot Drinks
* Quick Bites

## Technologies Used

* Python
* Dictionaries
* Loops
* Input handling

## How to Run

1. Clone the repository
2. Run the Python file

```bash
python cafe.py
```
